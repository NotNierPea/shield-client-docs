# How To Install Mods

### Mod Installation (Mod Manager)

You can use the in-game mod manager to install mods

### Mod Installation (Manually)

Extract the zip folder in `Call of Duty Black Ops 4\project-bo4\mods`, if the mod has zone files, then extract in `Call of Duty Black Ops 4\project-bo4` only.

\
\
_**Make sure to reload mods (in mod manager or shield options) after installing any mod.**_
