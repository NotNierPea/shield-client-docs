# How To Install

Download the launcher from [useful-links.md](../useful-infomation/useful-links.md "mention")

Open the `zip` file. then extract the contents of the zip file into your `Black Ops 4` folder.

<figure><img src="../.gitbook/assets/Screenshot 2024-01-03 201636.png" alt=""><figcaption></figcaption></figure>

Then open `Launch Project BO4.exe`.
